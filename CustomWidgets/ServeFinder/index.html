Serve Finder

This widget allows users to search for serve opportunities by various parameters. It displays a list of matching serve opportunities with details and a link to the opportunity page in a nested JSON format. Check out the Live JSON Output (or the console log) to see the object and refer to the github to see examples of how other widgets have been used in this manner (eg. Group Finder). The github for this project can be found here. The following parameters can be used to filter serve opportunities (all are optional):

  • @CongregationID - One congregation can be selected at a time
  • @OpportunityTypeID - One opportunity type can be selected at a time
  • @MinistryIDs - A comma separated list of ministry ID's can be passed
  • @Search - Search term to filter events by various fields in the event
  • @OpportunityIDs - Comma-separated list of specific event IDs to filter to
  • @Page - Page number for pagination
  • @NumPerPage - Number of events per page
  • @Username - Automatically passed by the Login Widget
