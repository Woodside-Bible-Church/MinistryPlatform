Event Finder

This widget allows users to search for events by various parameters. It displays a list of matching events with details and a link to the event page in a nested JSON format. Open the console log to see the object and refer to the github to see examples of how other widgets have been used in this manner (eg. Group Finder) The following parameters can be used to filter events (all are optional):

  • @CongregationID - One congregation can be selected at a time
  • @MinistryIDs - A comma separated list of ministry ID's can be passed
  • @Search - Search term to filter events by various fields in the event
  • @EventIDs - Comma-separated list of specific event IDs to filter to
  • @Page - Page number for pagination
  • @NumPerPage - Number of events per page
  • @Username - Automatically passed by the Login Widget
